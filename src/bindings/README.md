# Bindings Module

## Purpose
pybind11 bindings exposing LOBEnv to Python.

## Interface
- `lob_rl` Python module
- `LOBEnv` class with `reset()`, `step(action)`
- Zero-copy numpy arrays for observations

## Dependencies
- Depends on: `LOBEnv`
- Depended on by: Python training scripts

## Status
Not implemented.
